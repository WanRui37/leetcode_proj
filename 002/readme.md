# 字母异位词分组
给你一个字符串数组，请你将 字母异位词 组合在一起。可以按任意顺序返回结果列表。

## 方法一
- 排序: 将每个字符串排序后作为哈希表的键，相同键的字符串归为一组
- 具体实现: 
    - 遍历每个字符串，将其排序后作为key
    - 使用unordered_map存储key对应的字符串列表
    - 最后将哈希表中的所有值转换为结果
- 时间复杂度：O(n * k log k)，其中n是字符串数量，k是字符串最大长度
- 空间复杂度：O(n * k)

## 方法二
- 计数: 使用字符计数作为哈希表的键
- 具体实现: 
    - 统计每个字符串中每个字符的出现次数
    - 将计数结果转换为字符串格式作为key
    - 使用哈希表进行分组
- 时间复杂度：O(n * k)
- 空间复杂度：O(n * k)
