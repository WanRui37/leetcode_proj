# 盛最多水的容器

给定一个长度为`n`的整数数组 `height` 。有 `n` 条垂线，第 `i` 条线的两个端点是 `(i, 0)` 和 `(i, height[i])` 。

找出其中的两条线，使得它们与 `x` 轴共同构成的容器可以容纳最多的水。

返回容器可以储存的最大水量。


## 方法一
- 双指针: 使用左右指针从数组两端向中间移动
- 具体实现: 
    - 初始化左指针l=0，右指针r=数组长度-1
    - 计算当前容器的面积：min(height[l], height[r]) * (r - l)
    - 更新最大面积ans = max(ans, area)
    - 移动较矮的那一侧指针（如果height[l] <= height[r]则l++，否则r--）
- 时间复杂度：O(n)
- 空间复杂度：O(1)
